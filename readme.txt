MiniGit Program Functionality:
Users will be prompted to initialize a new repository. 
Users will be presented with a menu where they can choose to add a file, remove a file, commit changes, and checkout a previous commit.

Add a file:
Users can add files to the current singly linked list of the current doubly linked list node.

Remove a file:
Users can remove files from the current singly linked list of the current doubly linked list node.

Commit Changes:
Users can commit changes to a new doubly linked list node.

Checkout:
Users can checkout any previous commits and restore previous versions.
